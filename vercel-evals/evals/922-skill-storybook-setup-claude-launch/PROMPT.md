This project has Storybook configured and a `storybook` script in `package.json`.

Set up Claude so it can start this project's Storybook dev server: create or repair the
`.claude/launch.json` launch configuration with a Storybook entry. There is already an
existing `.claude/launch.json` with a non-Storybook launch entry; keep it intact.

Do not start Storybook yourself; only configure the launch entry.
